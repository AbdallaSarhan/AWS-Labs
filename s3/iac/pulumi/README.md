# Pulumi

## Installing Pulumi
```sh
curl -fsSL https://get.pulumi.com | sh
```
## Create new Pulumi example in Python
```sh
pulumi new aws-python
```
## Deploy the Stack and Resources
```sh
pulumi up
```
## Delete the Stack's Resources
```sh
pulumi destory
```
## Delete the Stack
```sh
pulumi stack rm
```
